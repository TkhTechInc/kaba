# Rules Discovery Process for AI Agents

**MANDATORY** — Execute at the start of EVERY conversation.

## Process

1. **Load global rules**: Read and apply `AI/global/rules/rules-bank.md`
2. **Load local rules** (if any): Read and apply any files in `AI/local/rules/`
3. **Review lessons**: If `tasks/lessons.md` exists and has content, read it to avoid repeating past mistakes

## Precedence

- Local rules supplement (not replace) global rules
- Lessons apply as corrective guidance
